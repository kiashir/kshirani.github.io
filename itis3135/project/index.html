Fires can be classified into five different categories based of the agent that fuel them :

  • Class A: Ordinary Combustables

  • Class B: Flammable Liquids

  • Class C: Electrical

  • Class D: Combustible Metals

  • Class F (EU/AU) / Class K (US): Cooking Oils and Fats

It is very important to use proper fire extinguishing methods and safety procedures depending on the combustion type in order to control a fire. In this website you will learn more about different classes of fire and proper ways of extinguishing/controlling them.
